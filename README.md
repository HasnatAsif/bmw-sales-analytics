# BMW Sales Analytics

## Project Overview
This project analyzes BMW sales data using SQL and Power BI to evaluate revenue performance, sales trends, and regional distribution.

## Tools Used
- SQL (SQLite)
- Power BI

## Data Preparation Steps
- Added surrogate primary key
- Created revenue metric (Price × Sales Volume)
- Built analytics-ready view

## Business Questions Answered
- Which BMW models generate the highest revenue?
- How does revenue vary across regions?
- What are sales trends over time?

## Dashboard Features
- Total Revenue KPI
- Total Units Sold KPI
- Average Price KPI
- Revenue by Model
- Revenue by Region
- Sales Trend by Year
- Fuel Type Filter
